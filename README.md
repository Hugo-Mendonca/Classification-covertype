# Classification-covertype
Nosso projeto consiste na classificação de cobertura florestal da Floresta Nacional de Roosevelt (EUA). Dataset com 581.012 instâncias e 54 variáveis (topográficas, geológicas e climáticas). Possui 7 classes alvo com desbalanceamento severo (classes raras &lt; 1%), exigindo métricas robustas como F1-Score Macro para modelos precisos.
