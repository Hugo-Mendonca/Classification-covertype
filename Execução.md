# Inventário Florestal: Classificação de Cobertura com Machine Learning

Este projeto apresenta uma investigação sistemática sobre a classificação de cobertura florestal utilizando técnicas de *Machine Learning*. O foco central é a comparação entre arquiteturas de modelos, variando desde classificadores lineares até *Ensembles* complexos (Stacking Heterogêneo), integrados a *pipelines* de pré-processamento para tratamento de desbalanceamento de classes.
